**1. Add `useState` to existing React import** (line 1):
Change `import { useMemo } from 'react'` to `import { useState, useMemo } from 'react'`.

**2. Add displayMode state** inside the component function, after the existing hooks (after line ~55):
```jsx
const [displayMode, setDisplayMode] = useState(
  () => localStorage.getItem('dashboard_display_mode') || 'usage'
);
```
Default is `'usage'` per D-03.

**3. Add toggle UI** — replace the heading `<motion.div variants={fadeUp}>` block (lines 161-164) with:
```jsx
<motion.div variants={fadeUp} className="flex items-center justify-between">
  <div>
    <h1 className="text-xl font-semibold text-foreground tracking-tight">{t('dashboard.title')}</h1>
    <p className="text-[13px] text-muted-foreground mt-1">{t('dashboard.description')}</p>
  </div>
  <div className="flex items-center gap-1 p-1 rounded-lg bg-muted w-fit">
    {['cost', 'usage'].map((mode) => (
      <motion.button
        key={mode}
        whileTap={{ scale: 0.98 }}
        onClick={() => {
          setDisplayMode(mode);
          localStorage.setItem('dashboard_display_mode', mode);
        }}
        className={[
          'px-3 py-1.5 rounded-md text-xs font-medium transition-colors duration-150',
          displayMode === mode
            ? 'bg-background text-foreground shadow-sm'
            : 'text-muted-foreground hover:text-foreground',
        ].join(' ')}
      >
        {t(mode === 'cost' ? 'dashboard.toggleCost' : 'dashboard.toggleUsage')}
      </motion.button>
    ))}
  </div>
</motion.div>
```
This places the heading left-aligned and toggle right-aligned in the same row per D-02. Toggle styling matches LanguageSwitcher pattern per UI-SPEC: `bg-muted` container, `bg-background shadow-sm` active pill, `text-xs font-medium` labels.

Change:

This switches between cost and usage display. Guards null `total_usage` with `|| 0` per Pitfall 2. Gets `unit` from `ctxSection` (already defined on line 192) per Pitfall 3. The "Total Spending" card (lines 167-189) is NOT modified — it always shows `formatCurrency(totalCost)` per D-05.
