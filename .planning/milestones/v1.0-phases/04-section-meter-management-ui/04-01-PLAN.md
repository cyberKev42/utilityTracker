Create `frontend/src/components/settings/MeterRow.jsx`:

**Props:** `meter` (object: { id, name }), `sectionId` (string), `isNew` (boolean, default false — when true, start in edit mode with autoFocus)

**Sortable integration:**
- Import `useSortable` from `@dnd-kit/sortable` and `CSS` from `@dnd-kit/utilities`
- Call `const { attributes, listeners, setNodeRef, transform, transition, isDragging } = useSortable({ id: meter.id })`
- Apply style: `{ transform: CSS.Transform.toString(transform), transition, opacity: isDragging ? 0.5 : 1 }`
- Do NOT use Framer Motion `layout` prop (conflicts with dnd-kit transforms per RESEARCH.md Pitfall 3)

**Layout:** Single row `flex items-center gap-2 px-3 py-2 rounded-md min-h-[44px] hover:bg-muted/50`
- Left: drag handle button with `{...attributes} {...listeners}` — render `HiOutlineBars3` icon (16px), `aria-label={t('settings.sections.reorderHint')}`
- Center: meter name (text or input depending on editing state)
- Right: delete button (hidden until hover via `opacity-0 group-hover:opacity-100 transition-opacity`) — render `HiOutlineTrash` icon (16px)

**Inline editing:**
- State: `const [editing, setEditing] = useState(isNew)` and `const [value, setValue] = useState(meter.name)`
- Display mode: `<span onClick={() => setEditing(true)} className="text-sm text-foreground cursor-pointer flex-1">{meter.name}</span>` — render meter.name directly, NOT through t()
- Edit mode: `<Input autoFocus value={value} onChange={e => setValue(e.target.value)} onBlur={handleSave} onKeyDown={e => e.key === 'Enter' && handleSave()} className="flex-1 h-8 text-sm" />`
- `handleSave`: if `value.trim()` differs from `meter.name` and is non-empty, call `await updateMeter(sectionId, meter.id, { name: value.trim() })`. Catch errors: revert `setValue(meter.name)` and set `error` state briefly. If value is empty or unchanged, revert and exit edit mode.
- Get `updateMeter` and `deleteMeter` from `useSections()` hook

**Delete confirmation:**
- State: `const [deleteOpen, setDeleteOpen] = useState(false)`
- Delete button click: `setDeleteOpen(true)`
- Confirmation dialog using Radix Dialog: title = `t('settings.sections.deleteMeterTitle')`, body = `t('settings.sections.deleteMeterMessage', { name: meter.name })`, confirm button = `t('settings.sections.deleteMeter')` with `variant="destructive"`, cancel button = "Discard Changes" with `variant="outline"`
- On confirm: `await deleteMeter(sectionId, meter.id)`, close dialog

**Error state:**
- If updateMeter or deleteMeter throws, show `text-xs text-destructive` inline error below the row for 4 seconds, then auto-dismiss via setTimeout
