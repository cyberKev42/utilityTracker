## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, chart label gaps |
| sm | 8px | Compact element spacing, tab bar gap between tabs |
| md | 16px | Default element spacing, card padding, section gaps |
| lg | 24px | Section padding, stat card internal spacing |
| xl | 32px | Layout gaps between chart blocks |
| 2xl | 48px | Major section breaks, page header to first content |
| 3xl | 64px | Page-level vertical spacing |

Exceptions:
- Touch targets (inputs, select controls, date fields): min-height 44px — established in `index.css` for `input[type="date"]`, `input[type="number"]`, `select`
- Chart Y-axis width: 52px (established in existing chart code — RESEARCH.md Pattern 5)

## Typography

**Primary visual anchor:** Summary Stat Cards row (top of page) — largest numeric displays draw the eye first.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body / tooltip value | 16px | 400 (regular) | 1.5 |
| Label / small UI / empty state body | 14px | 400 (regular) | 1.4 |
| Stat value (tabular) | 24px | 600 (semibold) | 1.2 |
| Heading / page title | 20px | 600 (semibold) | 1.2 |

Notes:
- Chart axis tick labels: 12px at weight 400 — smallest permitted size, used only for Recharts axis ticks
- Stat value suffix (unit label): 14px at weight 400, `text-muted-foreground` — e.g. "1,245 **kWh**" suffix; maps to Label row
- Tooltip primary value: 16px at weight 600 — maps to Body row
- `tabular-nums` class applied to all numeric stat displays for alignment
- `tracking-tight` on large numeric displays (24px stat values)

Weights in use: 400 (regular) and 600 (semibold) only. `font-bold` (700) is not used in this phase.

Source: `frontend/src/pages/StatisticsDetail.jsx` lines 71–75, RESEARCH.md Pattern 5 and Summary Stat Card example

## Component Specifications

### Usage/Cost Segmented Control

- **Placement:** Top-right of page header area, on both `Statistics.jsx` and `StatisticsDetail.jsx`
- **Structure:** Two-segment inline button group with shared rounded-lg border
- **Active state:** `bg-primary text-primary-foreground`
- **Inactive state:** `text-muted-foreground hover:text-foreground`
- **Size:** `px-3 py-1.5`, `text-xs font-medium`
- **Border:** `border border-border/40 rounded-lg overflow-hidden`
- **Labels:** "Usage" / "Cost" (i18n keys: `statistics.usage` / `statistics.cost`)
- **Default:** Usage selected

Source: RESEARCH.md Segmented Control code example (locked decision from CONTEXT.md)

### Meter Tab Bar

- **Placement:** Below the page title / section header, above charts — only shown when `section.meters.length > 1`
- **Structure:** Horizontal scrollable flex row of pill buttons
- **Tabs:** "All" tab first, then one tab per meter in order
- **Active state:** `bg-primary/10 text-primary`
- **Inactive state:** `text-muted-foreground hover:text-foreground`
- **Size:** `px-3 py-1.5 rounded-md text-xs font-medium whitespace-nowrap`
- **Gap between tabs:** 4px (gap-1)
- **Overflow:** `overflow-x-auto` (horizontal scroll on narrow viewports)
- **"All" tab label:** i18n key `statisticsDetail.allMeters`
- **Animation:** Framer Motion tab indicator — use existing `fadeUp` pattern for initial mount

Source: RESEARCH.md Meter Tab Bar code example (locked decision from CONTEXT.md)

### Granularity Toggle (SpendingLineChart)

- **Placement:** Inside `SpendingLineChart` component, top-right corner of the chart card
- **Segments:** "Daily" / "Weekly" / "Monthly" (3-way)
- **Default:** Weekly
- **Style:** Same segmented control pattern as Usage/Cost toggle, same sizing
- **i18n keys:** `statistics.granularity.daily` / `statistics.granularity.weekly` / `statistics.granularity.monthly`

Source: CONTEXT.md locked decision — "Weekly is the default granularity"

### Year Selector

- **Placement:** Top-right of `StatisticsDetail.jsx` header, left of Usage/Cost toggle
- **Style:** Arrow buttons (`<` / `>`) flanking the year number, or a `<select>` dropdown
- **Recommended:** Arrow buttons — `text-sm font-medium` year label between two icon buttons
- **Button size:** 32px × 32px touch target minimum
- **i18n:** Year number is numeric, no translation needed

Source: CONTEXT.md locked decision — "Year selector (dropdown or arrows) to switch between years"

### Summary Stat Cards (StatisticsDetail top row)

- **Layout:** 3 cards in a row (flex or grid-cols-3)
- **Stats:** Total usage + unit | Total cost | Average per day
- **Value typography:** 24px semibold (600), `tabular-nums`, `tracking-tight`
- **Unit suffix:** 14px regular, `text-muted-foreground`, inline after value
- **Label:** 12px regular, `text-muted-foreground`, above value
- **Card:** shadcn `<Card><CardContent>` with `p-4` padding
- **No entry count stat** (locked decision from CONTEXT.md)

Source: CONTEXT.md locked decision, RESEARCH.md Summary Stat Card code example

### Stacked Area Chart ("All" tab)

- **Component:** Recharts `<AreaChart>` with `<Area stackId="meters">` per meter
- **Fill opacity:** 0.6 per meter area
- **No dot markers:** `dot={false}` on all Areas
- **Colors:** SECTION_COLORS palette by meter index (see Color section above)
- **Total line:** Additional `<Area>` without `stackId`, no fill (`fill="none"`), single-color stroke showing aggregate — use `hsl(0 0% 93%)` (foreground) at opacity 0.5
- **Tooltip:** Custom component — shows TOTAL only (sum of all meter values), numeric-only, no unit label
- **Y-axis label:** Section unit string when in Usage mode; currency symbol when in Cost mode

Source: RESEARCH.md Pattern 3 (stacked area), CONTEXT.md locked decision — "Tooltips do NOT show units"

### Chart Tooltips (all charts)

- **Container:** `bg-card border border-border/50 rounded-lg px-3.5 py-2.5 shadow-xl backdrop-blur-sm`
- **Date label:** 12px regular, `text-muted-foreground`, `mb-1`
- **Value:** 16px semibold (600), `text-foreground tabular-nums tracking-tight`
- **No units in tooltip** (locked decision from CONTEXT.md)

Source: `frontend/src/pages/StatisticsDetail.jsx` lines 69–75

### Loading Skeleton

- **Pattern:** Use `animate-pulse` Tailwind utility on placeholder rects
- **Skeleton background:** `bg-muted rounded-md`
- **Stat card skeleton:** Three cards, each with two placeholder rects (label + value height)
- **Chart skeleton:** Full-width rect at chart height (approx 200px)
- **Tab bar skeleton:** Three pill-shaped rects at tab size

Source: Claude's discretion (established animate-pulse pattern, standard shadcn skeleton approach)

### Empty State (per meter tab)

- **Trigger:** Selected meter tab has zero entries
- **Layout:** Centered vertically in chart area, icon above text
- **Icon:** `HiOutlineChartBar` at 32px, `text-muted-foreground`
- **Heading:** "No data" (i18n key: `statisticsDetail.emptyMeter.heading`)
- **Body:** "No entries recorded for this meter yet." (i18n key: `statisticsDetail.emptyMeter.body`)
- **Typography:** Heading 14px semibold (600) `text-foreground`, body 14px regular `text-muted-foreground`
- **Min height:** 160px container so the chart area doesn't collapse

Source: Claude's discretion (locked: "Empty state handling per meter tab" is Claude's discretion per CONTEXT.md)

## Interaction States

| Component | States |
|-----------|--------|
| Meter tab button | default / hover / active (selected) |
| Usage/Cost toggle segment | default / hover / active (selected) |
| Granularity toggle segment | default / hover / active (selected) |
| Year arrow button | default / hover / disabled (at boundary year) |
| Stat card | static (no hover) |
| Chart area | hover (tooltip appears on data point) |

**Transition duration:** 150ms ease-out for all toggle/tab state changes (matches existing `cardHover` pattern: `duration: 0.15`)

## Animation Contract

All animation uses Framer Motion. Use the established patterns from `StatisticsDetail.jsx`:

| Pattern | Definition | Used on |
|---------|------------|---------|
| `fadeUp` | opacity 0→1, y 10→0, duration 0.25, ease [0.25, 0.1, 0.25, 1] | Stat cards, chart cards on mount |
| `stagger` | staggerChildren 0.08 | Wrapper container for stat cards |
| `cardHover` | y: -2, duration 0.15, easeOut | Not used on stat cards in this phase (read-only display) |

Tab switch: no page-level animation. Chart data transition: Recharts built-in `isAnimationActive` (default true, keep as-is).
