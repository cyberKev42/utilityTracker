1. Add `TouchSensor` to the import from `@dnd-kit/core`:

2. Update the `meterSensors = useSensors(...)` call (lines 55-57) to:
```javascript
const meterSensors = useSensors(
  useSensor(PointerSensor, {
    activationConstraint: { distance: 8 },
```
