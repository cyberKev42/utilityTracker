2. Verify a meter dropdown appears with meters from that section only
3. Select a usage-mode meter
4. Leave both dates as today (default)
5. Enter a usage amount (e.g. 50)
6. Click "Save entry" — expect success screen
7. Click "Add another" — expect ALL fields reset (section, meter, dates, amount)

**ENTR-02 — Date range split preview:**

**ENTR-03 — Meter reading mode:**
1. Select a section that has a reading-mode meter (check Settings if needed)

5. Enter a new reading higher than the last
6. Save — expect success

**ENTR-04 — First reading for new meter:**
1. In Settings, create a new meter with entry_mode = "reading" (or use one with no entries)
2. Go to /add-entry, select the section and that new meter
3. Verify: info banner appears: "This is the first reading for this meter..."
4. Enter a reading value and save
5. Verify success screen shows
